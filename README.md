O Filebrax Hub (cujo nome não guarda semelhança com meu nickname) é um projeto completo e funcional para realizar uploads e downloads de arquivos em um servidor. Foi desenvolvido utilizando o Flask como seu Micro Framework, com o objetivo de oferecer um ambiente interno eficiente. Além disso, graças à sua arquitetura organizada por meio de Blueprints, é possível integrar facilmente um módulo adicional de processamento no backend (e. g. Tracker Tickets, que é uma aplicação de rastreamento e gerenciamento que pretendo publicar em breve). :)
